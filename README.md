## 📋 نظرة عامة

**جمعية** هي منصة إدخار جماعي (ROSCA - Rotating Savings and Credit Association) موجهة للسوق الجزائري. تتيح للمستخدمين إنشاء والانضمام إلى جمعيات ادخار جماعية مع آلية آمنة وشفافة لتوزيع الأموال.

### ✨ المميزات الرئيسية

- 🔐 **نظام أمان متكامل**: تحقق من الهوية (KYC) وحماية البيانات
- 💰 **إدخار جماعي**: انضم لجمعيات بمبالغ شهرية مختلفة
- 📊 **تتبع مباشر**: متابعة تقدم الجمعيات والمدفوعات
- 🔔 **إشعارات ذكية**: تذكيرات بالدفع وتحديثات الحالة
- 📱 **تصميم متجاوب**: يعمل على جميع الأجهزة
- 🎨 **واجهة عربية**: دعم كامل للغة العربية وRTL

## 🚀 التقنية المستخدمة

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **State Management**: Zustand
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Storage**: LocalStorage (Persist)

## 🛠️ التثبيت والتشغيل

### المتطلبات

- Node.js 18+
- npm أو yarn

### خطوات التثبيت

1. **استنساخ المستودع**
```bash
git clone <repository-url>
cd jamiya-app
```

2. **تثبيت التبعيات**
```bash
npm install
```

3. **تشغيل خادم التطوير**
```bash
npm run dev
```

4. **فتح التطبيق**
```
http://localhost:5173
```

### البناء للإنتاج

```bash
npm run build
```

سيتم إنشاء الملفات في مجلد `dist/`.

## 📱 شاشات التطبيق

### 1. شاشة البداية (Splash)
- شعار التطبيق مع رسوم متحركة
- الانتقال التلقائي بعد 2.5 ثانية

### 2. تسجيل الدخول
- رقم الهاتف (+213 تلقائياً)
- كلمة المرور
- التحقق من صحة البيانات

### 3. إنشاء حساب (3 خطوات)
- **الخطوة 1**: رقم الهاتف والبريد الإلكتروني
- **الخطوة 2**: كلمة المرور (مع مؤشر القوة)
- **الخطوة 3**: الاسم الكامل ومستوى الدخل

### 4. التحقق من الهاتف (OTP)
- إدخال رمز مكون من 6 أرقام
- عداد تنازلي 60 ثانية
- **الرمز التجريبي**: `123456`

### 5. التحقق من الهوية (KYC)
- رفع صورة بطاقة التعريف
- رفع صورة سيلفي
- محاكاة معالجة 3 ثواني

### 6. لوحة التحكم الرئيسية
- إجمالي الادخار
- العضويات النشطة
- الجمعيات المتاحة
- إحصائيات سريعة

### 7. تفاصيل الجمعية
- معلومات شاملة عن الجمعية
- شريط تقدم
- مميزات وكيفية العمل
- زر الحجز

### 8. الحجز والدفع
- ملخص الحجز
- اختيار طريقة الدفع (بطاقة/CCP)
- رسوم التفعيل: 2,000 دج

### 9. نجاح الدفع
- رسالة تأكيد
- عداد تنازلي 48 ساعة
- خطوات التحقق

### 10. التحقق من البريد
- إدخال رقم معاملة البريد
- صيغة: `POST-YYYY-XXXX-NNNN`
- أمثلة صالحة متاحة

### 11. العضوية النشطة
- تفاصيل العضوية
- شريط تقدم الجمعية
- زر دفع المساهمة

### 12. سجل المعاملات
- فلترة حسب النوع
- إجمالي المساهمات والمستلم
- إيصالات قابلة للتحميل

### 13. الإشعارات
- إشعارات فورية
- تحديد الإشعارات غير المقروءة
- تعليم الكل كمقروء

### 14. الملف الشخصي
- معلومات المستخدم
- حالة التوثيق
- إعدادات الحساب
- تسجيل الخروج

### 15. شاشة الاكتمال (الاحتفال)
- تأثيرات احتفالية
- إحصائيات الجمعية
- شهادة رقمية

## 🔧 إدارة الحالة (State Management)

يستخدم التطبيق **Zustand** لإدارة الحالة مع التخزين المستمر في LocalStorage:

```typescript
// الحالة الرئيسية
interface AppState {
  currentUser: User | null;
  isAuthenticated: boolean;
  jamiyas: Jamiya[];
  bookings: Booking[];
  activeMemberships: ActiveMembership[];
  transactions: Transaction[];
  notifications: Notification[];
  currentScreen: ScreenType;
}
```
