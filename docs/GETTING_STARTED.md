# البدء السريع

دليل سريع للبدء مع تطبيق جمعية.

## المتطلبات

- Node.js 18+
- npm أو yarn
- Git

## التثبيت

### 1. استنساخ المستودع

```bash
git clone https://github.com/jamiya/jamiya-app.git
cd jamiya-app
```

### 2. تثبيت التبعيات

```bash
npm install
```

### 3. تشغيل خادم التطوير

```bash
npm run dev
```

### 4. فتح التطبيق

افتح المتصفح على: `http://localhost:5173`

## سيناريو الاستخدام

### 1. إنشاء حساب جديد

1. افتح التطبيق
2. اضغط "سجل الآن"
3. أدخل بياناتك
4. أكد رقم هاتفك بالـ OTP (123456)
5. أكمل التحقق من الهوية

### 2. الانضمام لجمعية

1. سجل الدخول
2. تصفح الجمعيات المتاحة
3. اختر جمعية
4. اضغط "احجز مكانك"
5. ادفع رسوم التفعيل (2,000 دج)
6. أدخل رقم البريد

### 3. دفع مساهمة

1. اذهب لـ "عضوياتك النشطة"
2. اختر الجمعية
3. اضغط "دفع الآن"

## الأوامر المتاحة

```bash
# التطوير
npm run dev

# البناء
npm run build

# المعاينة
npm run preview

# الفحص

# Docker
docker-compose up -d

# Makefile
make help
```
